# 카드 메이커

행사·클래스 홍보 카드를 쉽게 만드는 정적 웹사이트입니다.

## 기능

- 📷 이미지 업로드 및 위치 선택 (좌/우/상/하/배경)
- 🎨 배경색·강조색 자유 설정
- 🔤 한국어 폰트 6종 선택
- 📐 카드 크기 드래그 리사이즈 + 프리셋 (가로형·세로형·정사각·유튜브썸네일)
- 🧩 텍스트 블록 추가·삭제·순서 변경
- 💾 PNG로 저장 (2배 해상도)

## GitHub Pages 배포

1. 이 저장소를 GitHub에 업로드
2. Settings → Pages → Branch: `main`, folder: `/ (root)` → Save
3. `https://[username].github.io/[repo-name]` 으로 접속

## 파일 구조

```
index.html   — 메인 HTML
style.css    — 스타일
app.js       — 앱 로직
README.md    — 이 파일
```
